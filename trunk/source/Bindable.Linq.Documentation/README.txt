To debug/run:

1. Bring up project settings editor for Bindable.Linq.Documentation
2. Open Debug tab
3. Enter the following command line parameters:

..\..\..\Bindable.Linq\bin\debug\Bindable.Linq.dll Bindable.Linq.Extensions .\docs\

4. Ensure the Bindable.Linq project has been built.
5. Now you should be able to simply Debug\Run the Bindable.Linq.Documentation project when necessary.
6. Browse to source\Bindable.Linq.Documentation\bin\debug\docs\ to access the output.
